# Dragon-War-Proto
## Controls
| Action              | Key / Input                  | Notes                                      |
|---------------------|------------------------------|--------------------------------------------|
| Move                | WASD / Arrow keys            | Camera-relative movement                   |
| Sprint              | Left Shift / Right Shift     | Hold to move faster                        |
| Tail Attack (melee) | Space                        | Hold to keep animation looping if desired  |
| Normal Shoot        | Left Mouse Button (LMB)      | Hold to auto-fire after short delay        |
| Fly / Special Shoot | Right Mouse Button (RMB)     | Hold to auto-fire (different projectile)   |
| Aim                 | Move Mouse                   | Character rotates toward cursor position   |
